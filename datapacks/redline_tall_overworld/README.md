# Redline Tall Overworld Noise 4064

Datapack for Minecraft/NeoForge 26.2 tests with Redline Atlas Worldgen.

Overrides:

- `minecraft:dimension_type/overworld`
  - `min_y = -2032`
  - `height = 4064`
  - `logical_height = 4064`

- `minecraft:worldgen/noise_settings/overworld`
  - `noise.min_y = -2032`
  - `noise.height = 4064`
  - `sea_level = 0`
  - `aquifers_enabled = false`

Use on a new world. Old generated chunks will not regenerate.
