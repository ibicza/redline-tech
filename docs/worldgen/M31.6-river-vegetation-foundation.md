# M31.6 — River vegetation cleanup and loose-bed foundation

This patch is based on M31.5 (`02af71c`) from `redline-tech_2026-07-12_15-50-38.zip`.

## Fixed

### Floating vegetation above carved rivers

The post-generation river pass now clears natural replaceable features in every channel and bank column above the fitted profile before carving/filling:

- logs and leaves;
- flowers;
- short/tall grass and ferns;
- saplings, vines, snow layers and other replaceable worldgen vegetation.

The cleanup runs for both `CHANNEL` and `BANK` samples, so carving the terrain out from under a tree can no longer leave its trunk, crown or plants suspended over the river.

### Sand/gravel collapse

Sand and gravel are no longer used as their own structural foundation.

For every loose riverbed or loose bank cap:

1. a deterministic clay footing is built first;
2. its default thickness is 2–4 blocks per column;
3. the visible sand/gravel cap is placed last;
4. deeper edge containment uses stable sandstone/stone below the clay band.

This prevents riverbed gravel/sand from falling into caves or open spaces and leaving holes, item drops and broken water columns.

## New server config

Under `surface_polish`:

```toml
riverLooseFoundationMinBlocks = 2
riverLooseFoundationMaxBlocks = 4
```

The selected depth is deterministic from world X/Z, so regeneration of the same column produces the same foundation thickness.
