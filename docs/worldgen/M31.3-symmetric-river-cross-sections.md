# M31.3 — Symmetric river cross-sections

This patch makes each short centerline section authoritative for both sides of a river.

## Invariants

- Opposite columns with the same absolute lateral band use the same target Y.
- The complete section uses one water surface Y.
- The water surface is `waterBelowBankBlocks` below the symmetric rim; default is one block.
- The bed is at least one block below water and becomes deeper toward the center.
- Bed depth depends only on the quantized absolute distance to the centerline, so the bowl is mirrored.
- Replaceable terrain above an authoritative channel column is removed before source water is placed.
- The nearest bank columns are filled down to the maximum bed depth to seal caves and missing side walls.
- Downstream profile correction and junction joining remain lower-only.

## Cache

The cooked river fingerprint is now `river-profile-symmetric-cross-section-v5`. An older
`region-cache.rriver` is ignored and rebuilt automatically by `/rla reload`.

## Test

Use a new world or unfinished chunks, then run:

```text
/rla reload
/rla status
/tp @p -2041 300 -3238
/rla nearest_river 2048
/rla finish_here 4
```

The first reload can take time because HydroRIVERS, GLO-30 and WorldCover are cooked again.
