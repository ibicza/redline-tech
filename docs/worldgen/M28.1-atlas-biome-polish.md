# M28.1 — Atlas biome polish

## Цель

M28 уже добавил ESA WorldCover landcover и atlas biome resolver. M28.1 добивает первые практические проблемы:

- фактический biome теперь проще проверять через `/rla biome_sample`;
- biome guide получает seed из реального worldgen chunk context, а не `0L`;
- biome guide ограничен зарегистрированным worldgen chunk context и не лезет в чужие измерения;
- ESA WorldCover 10m проходит majority-filter перед выбором биома, чтобы не было 10-метровой мозаики;
- пороги температуры/влажности/поверхностного диапазона вынесены в config;
- `ResourceLocation` больше не используется, чтобы не падать на 26.2 API.

## Новые config-параметры

Секция `biome_guide`:

```toml
surfaceRelativeMinY = -48
landcoverSmoothRadiusBlocks = 48
landcoverSmoothStepBlocks = 24
ignoreWaterLandcover = true
freezingTemperatureC = 0.0
coldTemperatureC = 2.0
hotTemperatureC = 22.0
tropicalTemperatureC = 24.0
dryHumidity = 0.35
wetHumidity = 0.72
tropicalWetHumidity = 0.62
```

## Отладка

Команда:

```mcfunction
/rla biome_sample
```

теперь выводит и расчётный биом, и фактический biome storage:

```text
selected=ResourceKey[minecraft:worldgen/biome / minecraft:jagged_peaks]
actual=ResourceKey[minecraft:worldgen/biome / minecraft:jagged_peaks]
```

Если `selected` и `actual` отличаются, значит чанк был уже сгенерен старым кодом или biome source hook не применился в момент генерации этого чанка.

## Вода

`ignoreWaterLandcover=true` оставлен включённым по умолчанию. WorldCover water-пиксели пока не создают ocean/river/beach biomes. Водный контекст будет отдельным слоем, чтобы не вернуть проблему воды на склонах.
